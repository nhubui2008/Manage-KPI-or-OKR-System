                  ⚙️ BackEnd – Phạm Trần An An
                  Kiến Trúc & Kỹ Thuật
                  ASP.NET Core · EF Core · SQL Server · Gemini AI · Cookie Auth
                  🏗️ Kiến trúc Request Flow
                  1Browser gửi request
                  → Middleware: HTTPS, Auth, Antiforgery → Controller (routing theo action) → Service / Helper xử lý nghiệp vụ → EF Core → SQL Server
                  2Razor Views render giao diện trả về
                  🔒 Bảo mật & phân quyền
                  Cơ chế
                  [Authorize] → bắt buộc đăng nhập
                  [HasPermission("CODE")] → chặn theo quyền
                  Permission gán qua Role_Permission table
                  🗃️ ERD rút gọn
                  Foundation: Role, Permission, SystemUser, Employee, Department
                  OKR: MissionVision → OKR → KeyResult
                  KPI: EvalPeriod → KPI → KPIDetail → Assignment
                  CheckIn: KPICheckIn → CheckInDetail → History
                  Eval: EvaluationResult → GradingRank → BonusRule
                  Other: AuditLog, SystemAlert, AIHistory
                  🤖 AI & Services
                  Gemini API
                  GeminiService
                  AIAlertService
                  EmailService / OTP
                  EPPlus Excel
                  Google OAuth
